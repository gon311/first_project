# 📄 이력서 자동 첨삭 온라인 구인·구직 서비스

ChatGPT API를 활용하여 **이력서·자기소개서를 자동 분석하고 첨삭 피드백을 제공하는 웹 기반 구인·구직 서비스**입니다.

---

## 1. 프로젝트 개요

### 1.1 프로젝트 배경
- 구직자가 이력서 작성 과정에서 겪는 표현력·구조 문제 해결
- ChatGPT를 활용한 자연어 분석 및 자동 피드백 시스템 구현
- Spring 기반 웹서비스 아키텍처 설계 및 팀 협업 경험 확보

### 1.2 프로젝트 목표
- ChatGPT 연동 이력서 자동 첨삭 기능 구현
- MVC 패턴 기반의 안정적인 백엔드 구조 설계
- 사용자·채용공고·이력서 데이터를 통합 관리하는 구인·구직 서비스 구축
